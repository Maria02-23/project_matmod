Этап 2 Подходы, математический аппарат и алгоритмы

Аннотация

В отчёте представлены результаты второго этапа проекта по математическому моделированию образования планетной системы. Описана полная схема численного моделирования: инициализация протопланетного диска с кеплеровским вращением, вычисление гравитационных, контактных и диссипативных сил, интегрирование уравнений движения методом скоростного алгоритма Верле, процедура аккреции (слияния) частиц, а также методы ускорения вычислений (Particle-Mesh и быстрое мультипольное разложение FMM). Приведены блок-схема алгоритма и критерии контроля правильности расчётов.

1 Общая структура алгоритма

Моделирование эволюции протопланетного диска осуществляется методом пошагового численного интегрирования уравнений движения системы из взаимодействующих частиц. Временная эволюция системы разбивается на дискретные шаги длиной Δt. На каждом шаге последовательно выполняются вычисления сил взаимодействия, обновление кинематических параметров частиц и обработка столкновений.

Общая схема алгоритма на один шаг по времени:

  1. Вычисление всех сил, действующих на каждую частицу (гравитация, упругое отталкивание, трение).
  2. Интегрирование уравнений движения – обновление положений и скоростей.
  3. Обработка столкновений – проверка условий слияния (аккреции) и объединение частиц.
  4. Обновление вспомогательных структур (деревья, сетки для ускорения вычислений).
  5. Контроль сохранения величин (энергия, момент импульса) и вывод данных.

Алгоритм повторяется до достижения заданного физического времени либо до стабилизации системы.

2 Инициализация системы

Прежде чем запускать расчёт, нужно смоделировать начальный газопылевой диск.

2.1 Генерация начальных положений

В начальный момент частицы равномерно заполняют круг радиуса R₀ (протопланетный диск). Для получения случайной точки внутри круга используется следующий приём: генерируются два случайных числа ξ₁ и ξ₂ в интервале (0,1], радиус точки берётся как r = R₀·√ξ₁ (квадратный корень нужен для постоянства плотности по площади), азимутальный угол α = 2π·ξ₂, координаты x = r·cosα, y = r·sinα. В трёхмерном варианте добавляется небольшой разброс по вертикали z, чтобы диск имел малую толщину.

2.2 Начальные скорости (кеплеровское вращение)

Диск вращается вокруг центрального тела (прото-Солнца). Скорость каждой частицы соответствует круговой орбите под действием гравитации центральной массы. Согласно третьему закону Кеплера, угловая скорость ω убывает с расстоянием как ω(r) = ω₀·(R₀/r)^{3/2}. В декартовых координатах для частицы в точке (x,y):

\[ v_x = -\omega(r) \cdot y = -y \cdot \omega_0 \left(\frac{R_0}{r}\right)^{3/2} \]

\[ v_y = \omega(r) \cdot x = x \cdot \omega_0 \left(\frac{R_0}{r}\right)^{3/2} \]

\[ v_z = 0 \]

3 Вычисление сил

На каждую частицу действуют три типа сил: гравитационное притяжение, упругое отталкивание при соприкосновении и сила трения (диссипация).

3.1 Гравитационное взаимодействие

Гравитационная сила между двумя частицами вычисляется по закону Ньютона. В программе это реализуется двойным циклом по всем парам частиц. Для каждой пары находится вектор разности координат, его длина, затем вычисляется сила. Чтобы избежать численного взрыва при очень малых расстояниях, в знаменатель добавляется малая величина сглаживания ε (softening).

\[ \mathbf{F}_{ij}^{\text{grav}} = \gamma m_i m_j \frac{\mathbf{r}_j - \mathbf{r}_i}{(r_{ij}^2 + \varepsilon^2)^{3/2}} \]

3.2 Упругое отталкивание (при контакте)

Пусть a = R_i + R_j — сумма радиусов частиц, b = |r_j - r_i| — текущее расстояние между центрами. Если b < a, частицы проникли друг в друга, и включается сила упругого отталкивания:

\[ \mathbf{F}_{ij}^{\text{rep}} = k \left[ \left(\frac{a}{b}\right)^8 - 1 \right] \frac{\mathbf{r}_{ij}}{b} \]

Показатель степени 8 выбран так, чтобы сила была заметна только при реальном контакте. Сила направлена вдоль линии, соединяющей центры.

3.3 Сила трения (тангенциальная)

Для моделирования диссипации энергии используется тангенциальная сила, перпендикулярная линии центров. Её величина пропорциональна силе отталкивания и относительной тангенциальной скорости (с учётом вращения самих частиц):

\[ \mathbf{F}_{ij}^{\text{fric}} = \beta W_{\perp} F^{\text{rep}}(b) \cdot \mathbf{n} \]

где β — коэффициент интенсивности диссипации, W_⟂ — относительная тангенциальная скорость. Эта сила также создаёт момент, раскручивающий или замедляющий вращение каждой частицы.

4 Интегрирование уравнений движения (алгоритм Верле)

Второй закон Ньютона — дифференциальное уравнение, которое невозможно решить аналитически для системы многих частиц. Поэтому время разбивается на малые шаги Δt, и используется численный метод. Наиболее подходящим для гравитационных задач является скоростной алгоритм Верле.

Пусть в момент t_n известны положения r_n, скорости v_n и ускорения a_n (вычисленные через сумму сил). Тогда:

  1. Предсказание новых положений: r_{n+1} = r_n + v_n Δt + ½ a_n (Δt)²
  2. Вычисление новых ускорений a_{n+1} по положениям r_{n+1}
  3. Коррекция скоростей: v_{n+1} = v_n + ½ (a_n + a_{n+1}) Δt

Аналогично обновляются угловые скорости ω_i с использованием моментов сил. Этот метод обладает хорошей устойчивостью и сохраняет энергию даже при относительно больших Δt.

5 Обработка слияния частиц (аккреция)

Когда две частицы сближаются на расстояние меньше суммы их радиусов, они могут не просто оттолкнуться, а слиться в одну. В модели слияние происходит, если выполнено условие контакта и относительная скорость не превышает порога (чтобы не моделировать разрушение). При слиянии:

  • Масса новой частицы: m_new = m_i + m_j
  • Радиус: R_new = ∛(R_i³ + R_j³)
  • Положение (центр масс): r_new = (m_i r_i + m_j r_j) / m_new
  • Скорость (центр масс): v_new = (m_i v_i + m_j v_j) / m_new
  • Угловая скорость находится из закона сохранения полного момента импульса (орбитального и собственного).

После каждого шага интегрирования просматриваются все пары частиц. При обнаружении перекрытия пара помечается на слияние. Чтобы избежать «многоженства», пары сортируются по расстоянию и обрабатываются в порядке возрастания.

6 Алгоритмы ускорения вычислений

Прямой перебор всех пар даёт сложность O(N²). При N=10⁴ это уже 10⁸ операций на шаг, что очень медленно. Для ускорения применяются специальные методы.

6.1 Метод «частица–сетка» (Particle-Mesh)

Идея: заменить попарные силы решением уравнения Пуассона для гравитационного потенциала на равномерной сетке. Алгоритм: распределить массы частиц по узлам сетки → быстрым преобразованием Фурье (FFT) решить уравнение ∇²Φ = 4πγρ → найти силу как градиент потенциала на сетке → интерполировать силу обратно на частицы. Сложность снижается до O(M log M), где M — число узлов сетки.

6.2 Быстрое мультипольное разложение (FMM)

Этот метод считается «золотым стандартом» для гравитационных N-тел. Пространство рекурсивно разбивается на ячейки (квадродерево в 2D, октодерево в 3D). Для каждой ячейки вычисляются мультипольные моменты (суммарная масса, диполь, квадруполь...). При вычислении силы на частицу: если ячейка далеко — её вклад аппроксимируется несколькими первыми моментами; если близко — ячейка раскрывается на дочерние вплоть до отдельных частиц. Сложность FMM составляет O(N) или O(N log N), метод сохраняет высокую точность и хорошо работает при сильной кластеризации.

7 Выбор шага по времени

Шаг Δt должен быть достаточно мал, чтобы: частица за один шаг не пролетала больше своего радиуса; угол поворота по орбите был небольшим (обычно Δt < 0.01·T_min, где T_min — самый короткий период обращения); в процессе контактного взаимодействия не происходило чрезмерного сжатия. На практике выбирается адаптивный шаг: в начале расчёта вычисляются минимальный орбитальный период и минимальное время контакта, затем берётся наименьшее значение, умноженное на коэффициент надёжности (например, 0.1).

8 Контроль правильности и вывод результатов

Для проверки физической корректности модели на каждом шаге (или через несколько шагов) вычисляются:

  • полная механическая энергия (сумма кинетической, вращательной, гравитационной потенциальной и энергии упругого отталкивания);
  • полный момент импульса;
  • количество частиц, максимальная масса, размеры сгустков.

Если энергия начинает резко расти или падать — это признак численной нестабильности, и шаг Δt необходимо уменьшить. Все эти величины, а также координаты частиц записываются в файл для последующей визуализации и анализа.

9 Общая блок-схема алгоритма

  1. Задать параметры: N, R₀, ω₀, массы, плотность, коэффициенты k, β.
  2. Сгенерировать начальные положения (равномерно по диску).
  3. Задать кеплеровские скорости.
  4. Вычислить начальные ускорения (гравитация + контакты).
  5. Цикл пока t < t_max:
    1. Для каждой частицы: предсказать новое положение по скоростному Верле.
    2. По новым положениям вычислить все ускорения.
    3. Обновить скорости и угловые скорости.
    4. Обработать слияния (объединить перекрывающиеся частицы).
    5. Перестроить вспомогательные структуры (дерево FMM или сетку).
    6. t = t + Δt.
    7. Если шаг кратен заданному интервалу: записать энергию, момент импульса, координаты.

10 Заключение

На втором этапе проекта разработана полная схема численного моделирования, включающая: инициализацию диска с кеплеровским вращением; вычисление гравитационных, контактных и диссипативных сил; интегрирование по скоростному алгоритму Верле; процедуру аккреции (слияния) частиц; методы ускорения (Particle-Mesh или FMM) для работы с большим числом частиц. Предложенная схема является основой для программной реализации на третьем этапе проекта.
